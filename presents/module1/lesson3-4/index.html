Ветки - это указатели

Создали ветку

Находимся в ветке master

Сокращённый жупнал коммитов. Можно посмотреть где мы сейчас находимся.

Переходим на ветку testing


                        git checkout testing

Вносим изменения и комитим их


                        git commit -a -m 'made a change'

Возращаемся на ветку master


                        git checkout master

Вносим изменения в ветке master


                        git commit -a -m 'made other changes'

Смотрим на отображение ветвления в логах

Создаём и сразу переключаемя в новую ветку для решения задачи.


                        git checkout -b iss53
